Spacecraft Technical Documentation

System Requirements

Spacecraft is written in C# 3.0 (Microsoft Visual Studio 2008) and so requires the .NET Framework version 3.5 or greater to be installed on the host machine. (Or an equivalent, such as Project Mono.) The memory and disk space requires of Spacecraft depend on the size of the map being used, although the program itself takes less than 10MB of RAM and 2MB of disk space as of writing.

Installation

Spacecraft's executable requires read and write access to the directory it is placed in, and read and write access to the port specified in the configuration file. (Defaulting to 25565) Other than these, Spacecraft requires no special permissions and should run sucessfully from anywhere.

Configuration Options

When Spacecraft initializes, it creates the file properties.txt in the local directory. This file contains user-configurable settings for Spacecraft, specified in the format 

name = value
. Lines beginning with the hash symbol, #, are treated as comments and ignored. The full list of valid names follows. Invalid names are ignored.
  • heartbeat - "true" or "false". Whether to announce the server's precense to Minecraft.net.
  • http-port - Number, no commas, spaces, etc. The port on which to listen for HTTP connections. Defaults to port + 1.
  • max-players - Number. Maximum amount of players that can connect concurrently.
  • motd - String less than 64 characters long. The message to display to incoming players.
  • port - Number, no commas, spaces, etc. The port on which to listen for incoming players. Defaults to 25565.
  • public - "true" or "false". Whether the server is publically accessable.
  • server-name - String less than 64 characters long. The name of the server.
  • verify-names - "true" or "false". Whether to verify the names of incoming players with Minecraft.net, and reject unverified names. Defaults to true.
